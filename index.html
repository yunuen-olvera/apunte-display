Display

Definición

Pantalla, visualización, muestra. Lo que se muestra en la pantalla

La propieddad "display" especifica si un elemento es tratado como block o inline element y el diseño de lxs hijxs.


Display None

Permite dejar de visualizar contenido dentro del sitio.

Cuando ocupamos uno de estos, las etiquetas dejan de verse pero el código sigue ahí.

Display Block

El elemento genera un cuadro de elemento de bloque.

Esto quiere decir que el bloque (todo en HTML es un bloque) ocupa todo el espacio disponible de izquierda a derecha (todo el renglón dentro del componente padre).

Display Inline

El elemento genera uno o más cuadros de elemento en línea.

Significa que el contenedor o bloque se adapta al ancho de su contenido hijx.

Si le pones tamaño, ignora las propiedades de alto (height) y ancho (width).

El contenido solo ocupa el espacio que necesita.

Display Inline-Block

Si se le pone tamaño al display block funciona pero sigue ocupando todo el renglón.

El contenido ocupa el espacio designado.

Esto puede ser problemático si se quiere controlar el tamaño del componente y que el contenido esté en la misma línea.


NINGUNA DE LAS PROPIEDADES SE HEREDA

Esto funciona para los "padres". Flex y Grid funcionan para los "hijxs".
